### 1. Mac通过brew安装Redis
```
1. home-brew安装
  brew install redis
2. 启动Redis服务
  brew services start redis
3. 关闭Redis服务
  brew services stop redis
4. 重启Redis服务
  brew services restart redis
5. 启动Redis客户端
  redis-cli
```
